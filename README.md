# Meme Dev Mode 🚀

Bring some "energy" to your coding sessions with **Meme Dev Mode**! This extension adds humorous reactions and syntax highlights based on popular memes.

## Features

- **Error Forgiveness**: Type `maaf kar de bhai` to get instant forgiveness from the editor.
- **Ayenn Swag**: Type `ayenn function()` to activate functions with maximum style.
- **Baingan Energy**: The word `baingan` is automatically highlighted in purple with a 🍆 emoji and custom tooltip.
- **Meme Energy Status**: Check your current meme energy level in the status bar.
- **Toggle Mode**: Use the command `Meme Dev Mode: Toggle` or click the status bar item to enable/disable the fun.

## Supported Keywords

- `maaf kar de bhai`: Shows a "Error forgiven 😌" notification.
- `ayenn function()`: Shows a "Function activated, ayenn! 🚀" notification and hover.
- `baingan`: Highlights the word and shows a "Baingan Energy! 🍆" hover.

## Installation

1. Open the project in VS Code.
2. Press `F5` to open a new window with the extension loaded.
3. Start typing the meme keywords in any file!

## Packaging and Publishing

To package the extension for the marketplace:

1. Install `vsce`: `npm install -g @vscode/vsce`
2. Run `vsce package` to create a `.vsix` file.
3. Publish using `vsce publish` (requires a publisher account).
